Class: Basket

Basket()

Class permettant de gérer le panier

Constructor

new Basket()

Le constructor crée un objet javascript à partir du localStorage Cet objet peut être null si le localStorage n'existe pas on crée ensuite un objet "basket" contenant les données du panier en utilisant le model Basket Et pour finir on appel la méthode "countTotalArticles" pour afficher le nombre de produits.
