Confirmation()

Le constructor crée un objet javascript à partir du localStorage Cet objet peut être null si le localStorage n'existe pas Si le panier a été confirmé on désactive le bouton "panier" et on affiche le bouton "ma commande" Dans le cas contraire on passe le status de la confirmation à "false"
